# MyWebPortal

Название: Multi-Web Portal (JS Dashboard)

Описание:
Многофункциональный веб-портал, объединяющий более 20 мини-приложений и браузерных игр в едином интерфейсе. Проект реализован на чистом HTML, CSS и JavaScript без использования фреймворков.

Ключевые особенности:

Инструменты продуктивности: Список задач (LocalStorage), Помодоро-таймер, Конвертер валют (API), Погода (OpenWeather API).
Игровая зона: 12 классических игр, включая Змейку, Арканоид, Саймон, Flappy Bird и Викторину.
Мультимедиа: Генератор мемов (Canvas), Пианино и Драм-пад (Web Audio API).
UI/UX: Адаптивный дизайн в стиле Glassmorphism, система вкладок и анимации.
